## 🚀 Quick Start Commands

### Installation
```bash
npm install
cd ios && pod install && cd ..  # iOS only
```

### Run App
```bash
npm run android  # Android
npm run ios      # iOS (macOS only)
npm start        # Start Metro Bundler
```
